# 아기 이름 생성기

## 개요

이 프로젝트는 사용자에게 성별에 따라 아기 이름을 제안하는 간단하고 직관적인 웹 애플리케이션입니다.
다크 모드와 라이트 모드 기능을 제공하여 사용자 환경에 맞게 화면을 전환할 수 있습니다.

## 디자인 및 기능

*   **핵심 기능**:
    *   성별 선택 (남자, 여자, 또는 모두).
    *   "이름 생성" 버튼 클릭 시, 선택된 성별에 따라 이름 목록에서 무작위로 이름을 선택하여 표시.
    *   생성된 이름이 화면에 부드럽게 나타나는 애니메이션 효과.
    *   **제휴 문의 폼**: Formspree를 연동하여 사용자가 이름, 이메일, 메시지를 통해 제휴 문의를 남길 수 있는 기능.

*   **디자인**:
    *   **레이아웃**: 중앙 정렬된 컨테이너를 사용하여 모든 기기에서 콘텐츠가 깔끔하게 보이도록 합니다.
    *   **색상 (라이트 모드)**: 부드러운 파스텔 톤의 배경과 대비되는 텍스트 색상을 사용하여 가독성을 높입니다.
    *   **색상 (다크 모드)**: 어두운 배경과 밝은 텍스트를 사용하여 눈의 피로를 줄입니다.
    *   **글꼴**: 웹폰트를 사용하여 현대적이고 부드러운 느낌을 줍니다.
    *   **애니메이션**: 버튼 호버 효과와 이름이 나타날 때 부드러운 페이드 인 효과를 추가하여 사용자 경험을 향상시킵니다.
    *   **반응형**: 모바일, 태블릿, 데스크톱 등 다양한 화면 크기에 맞춰 레이아웃이 유연하게 변경됩니다.

## 구현 계획

1.  **HTML (`index.html`)**:
    *   테마 전환을 위한 버튼(아이콘 포함)을 추가합니다.
    *   기존 레이아웃을 유지하면서 테마 전환 버튼을 적절한 위치(예: 우측 상단)에 배치합니다.

2.  **CSS (`style.css`)**:
    *   CSS 변수(`--bg-color`, `--text-color`, `--container-bg`, `--accent-color` 등)를 도입하여 테마 관리를 용이하게 합니다.
    *   `:root`에 기본 라이트 모드 변수를 정의합니다.
    *   `.dark-mode` 클래스에 다크 모드용 변수를 정의합니다.
    *   기존 스타일을 CSS 변수를 사용하도록 리팩토링합니다.

3.  **JavaScript (`main.js`)**:
    *   테마 전환 버튼에 클릭 이벤트 리스너를 추가합니다.
    *   사용자의 테마 선택을 `localStorage`에 저장하여 페이지 새로고침 후에도 유지되도록 합니다.
    *   페이지 로드 시 `localStorage`에서 저장된 테마를 확인하고 적용합니다.

4.  **제휴 문의 폼 연동**:
    *   Formspree 엔드포인트(`https://formspree.io/f/xnjgkylq`)를 사용하여 문의 폼을 연동합니다.
    *   사용자 입력값(이름, 이메일, 메시지)에 대한 유효성 검사(HTML5 기본 기능 사용)를 수행합니다.

5.  **배포 (Deployment)**:
    *   `git push`를 통해 원격 저장소에 변경 사항을 배포합니다.
