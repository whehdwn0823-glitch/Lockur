# 아기 이름 생성기

## 개요

이 프로젝트는 사용자에게 성별에 따라 아기 이름을 제안하는 간단하고 직관적인 웹 애플리케이션입니다.

## 디자인 및 기능

*   **핵심 기능**:
    *   성별 선택 (남자, 여자, 또는 모두).
    *   "이름 생성" 버튼 클릭 시, 선택된 성별에 따라 이름 목록에서 무작위로 이름을 선택하여 표시.
    *   생성된 이름이 화면에 부드럽게 나타나는 애니메이션 효과.

*   **디자인**:
    *   **레이아웃**: 중앙 정렬된 컨테이너를 사용하여 모든 기기에서 콘텐츠가 깔끔하게 보이도록 합니다.
    *   **색상**: 부드러운 파스텔 톤의 배경과 대비되는 텍스트 색상을 사용하여 가독성을 높입니다.
    *   **글꼴**: 웹폰트를 사용하여 현대적이고 부드러운 느낌을 줍니다.
    *   **애니메이션**: 버튼 호버 효과와 이름이 나타날 때 부드러운 페이드 인 효과를 추가하여 사용자 경험을 향상시킵니다.
    *   **반응형**: 모바일, 태블릿, 데스크톱 등 다양한 화면 크기에 맞춰 레이아웃이 유연하게 변경됩니다.

## 구현 계획

1.  **HTML (`index.html`)**:
    *   페이지 제목과 설명을 추가합니다.
    *   성별 선택을 위한 라디오 버튼 그룹을 만듭니다.
    *   이름 생성을 위한 버튼을 추가합니다.
    *   생성된 이름이 표시될 영역을 만듭니다.

2.  **CSS (`style.css`)**:
    *   전체 페이지의 배경색과 글꼴을 설정합니다.
    *   메인 컨테이너의 스타일을 지정하여 콘텐츠를 중앙에 배치합니다.
    *   버튼과 라디오 버튼에 대한 맞춤 스타일을 적용합니다.
    *   이름이 표시될 영역의 스타일을 지정합니다.
    *   미디어 쿼리를 사용하여 반응형 디자인을 구현합니다.

3.  **JavaScript (`main.js`)**:
    *   남자 아기 이름과 여자 아기 이름 배열을 각각 만듭니다.
    *   DOM 요소를 가져옵니다 (버튼, 라디오 버튼, 이름 표시 영역).
    *   버튼 클릭 이벤트 리스너를 추가합니다.
    *   이벤트 리스너 함수 내에서 다음을 수행합니다:
        *   선택된 성별 값을 가져옵니다.
        *   성별에 따라 이름 목록을 필터링합니다.
        *   필터링된 목록에서 무작위로 이름을 선택합니다.
        *   선택된 이름을 이름 표시 영역에 업데이트합니다.
